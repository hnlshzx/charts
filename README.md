XCL-Charts
==========

<b>XCL-Charts v0.1</b><br/>
 &nbsp;&nbsp;&nbsp;&nbsp;Android开源图表库,基于原生的Canvas来绘制各种图表。作为一个图表库,<b>XCL-Charts</b>希望能在保证开发效率的同时，给程序员足够多的定制化能力。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;目前支持<b>3D/非3D柱形图、3D/非3D饼图、堆叠图、面积图、折线图、曲线图、
 环形图、南丁格尔玫瑰图、仪表盘、雷达图、圆形图</b>等，并支持<b>图表的缩放，手势移动，高密度柱形显示，期望线(分界线)定制，多图表的混合显示及同数据源不同类型图表切换及其它强大简明的定制能力。</b><br/>
